## WireBoard v5.5.3

### 주요 변경사항

**현황판 탭 신설 (기본 화면)**
- pcap 업로드 후 통계 현황판이 첫 화면으로 표시됩니다
- 전체 세션 수, 고유 IP, 공격 탐지, RST 패킷, 재전송 통계 카드
- 프로토콜 분포 및 통신 상태(정상/주의/이상) 요약

**세션 탐색기 (SessionExplorer)**
- IP 주소 검색으로 특정 IP 관련 세션만 필터링
- 비정상 세션만 표시 토글
- 세션 목록: 헬스 점수(0-100), 색상 구분, IP/포트 경로, RTT, 재전송 횟수
- 세션 클릭 → 핸드셰이크 상태/RTT/재전송/종료 유형/진단 문제/권장 조치
- 패킷 분석 열기 버튼으로 FlowViewer 연동

### 버그 수정 (v5.5.2에서 포함)
- 패킷/흐름 데이터가 세션 ID 불일치로 조회되지 않던 문제 해결
